[X](https://x.com/) (formerly Twitter) is a social media platform. This skill enables your agent to search tweets, read user profiles, browse timelines, and explore follower relationships.

## Required Environment

| Name             | Type   | Description             |
| ---------------- | ------ | ----------------------- |
| `X_TOKEN` | secret | X API v2 bearer token   |

Connect your X account in **Settings > Connectors** on the vm0 platform. The token is automatically injected at runtime.

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/x # [!code highlight]
```

## Run

Store your secret on the platform (recommended, one-time setup):

```bash
vm0 secret set X_TOKEN --body "your-x-token"
```

Then run your agent - secrets are automatically loaded:

```bash
vm0 run my-agent "find recent tweets about AI agents"
```

<Callout type="info">
For CI/CD or temporary overrides, pass secrets at runtime: `--secrets X_TOKEN=value`. See [Environment Variables](/docs/core-concept/environment-variable) for details.
</Callout>
