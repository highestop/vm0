[Xero](https://www.xero.com/) is a cloud-based accounting platform. This skill enables your agent to manage invoices, contacts, bank transactions, payments, and financial reports.

## Required Environment

| Name         | Type   | Description                |
| ------------ | ------ | -------------------------- |
| `XERO_TOKEN` | secret | Xero API bearer token      |

Connect your Xero account in **Settings > Connectors** on the vm0 platform. The token is automatically injected at runtime.

<Callout type="warn">
Xero API requires a `xero-tenant-id` header for all organisation-scoped requests. Call the `/connections` endpoint first to retrieve your tenant ID.
</Callout>

## Configuration

```yaml title="vm0.yaml"
version: "1.0"

agents:
  my-agent:
    framework: claude-code
    skills:
      - https://github.com/vm0-ai/vm0-skills/tree/main/xero # [!code highlight]
```

## Run

Store your secret on the platform (recommended, one-time setup):

```bash
vm0 secret set XERO_TOKEN your-xero-token
```

Then run your agent - secrets are automatically loaded:

```bash
vm0 run my-agent "list unpaid invoices"
```

<Callout type="info">
For CI/CD or temporary overrides, pass secrets at runtime: `--secrets XERO_TOKEN=value`. See [Environment Variables](/docs/core-concept/environment-variable) for details.
</Callout>
